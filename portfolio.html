Web Development

Burried within the pages of Dalhousie Unversity's new site, are some templates I've worked on. And some authentication backend scripts I've written. None of them I'm particularly proud enough of to put on this page. I did implement Jasig CAS server to handle centralized authenitcaiton but somewhere along the lines it never got used :( and kil-cas-1.ucis.dal.ca times out when pinged so I guess the project is done. N&S Tools is still up. It's a rather bare site using a template I worked on. They're using iframs in it. yuck. I believe the template is being used for the current login page of mydal, once you're inside it's uportal and more yuck.

The idea was to have the template look like dalonline once users logged in. but have the same features as the green template used for the main www.dal.ca pages. When I was done I merged them into one supertemplate for generating dal like pages. The login page was a priority for the new centralized login, that never ended up being used.
